Day 2 上午核心材料

工具实操：WorkBuddy 主力，Codex 对照

工具实操课的目标不是让每个人记住安装命令，而是让老师知道 WorkBuddy、Claude Code、Codex、GitHub 各自负责什么，出了错该看网络、账号、API Key 还是工具权限。

工具顺序

先选入口，再讲底层

工具分工

WorkBuddy 是主力，Codex 是工程化对照

不要把工具讲成排名。对老师来说，关键是任务匹配：科研文档先找 WorkBuddy，代码和文件级修改再考虑 Claude Code / Codex。

工具适合什么课堂定位
WorkBuddy科研任务、文档任务、流程编排、非程序员友好操作主力工具；优先用于老师日常科研效率。
Claude Code终端里的 Agent，能读文件、改文件、跑命令用于理解 Agent 工具和 DeepSeek API 接入。
Codex更工程化的代码 Agent，适合项目文件修改和验证作为 WorkBuddy 的工程化平替与对照。
Git / GitHub保存检查点、同步成果、追踪版本辅助能力；不展开复杂分支。
GitHub Pages把静态网页发布到公网可选展示方式，不是最终必交成果。
Claude Code + DeepSeek

用来理解终端 Agent 和 API 接入

Claude Code 是终端里的 Agent，能读文件、改文件、跑命令。DeepSeek API 接入只讲必要配置、安全边界和常见错误，不把它变成安装排雷大会。

Windows PowerShell
$env:ANTHROPIC_BASE_URL="https://api.deepseek.com/anthropic"
$env:ANTHROPIC_AUTH_TOKEN="<你的 DeepSeek API Key>"
$env:ANTHROPIC_MODEL="deepseek-v4-pro[1m]"
$env:ANTHROPIC_DEFAULT_OPUS_MODEL="deepseek-v4-pro[1m]"
$env:ANTHROPIC_DEFAULT_SONNET_MODEL="deepseek-v4-pro[1m]"
$env:ANTHROPIC_DEFAULT_HAIKU_MODEL="deepseek-v4-flash"
$env:CLAUDE_CODE_SUBAGENT_MODEL="deepseek-v4-flash"
$env:CLAUDE_CODE_EFFORT_LEVEL="max"
claude --version
claude

API Key 安全红线

  • 不截图发群。
  • 不写进 index.html、README 或课件截图。
  • 不提交到 GitHub。
  • 报错时可以贴错误码，不要贴完整 key。
WorkBuddy / Codex 对照

一个更像科研工作台，一个更像工程工具箱

WorkBuddy 更适合非程序员老师做文献、写作、数据和流程编排；Codex 更适合代码项目、文件级修改和自动验证。

WorkBuddy：老师优先入口

先用 Ask / Plan / Craft 或对应模式处理科研文档、课程材料、数据解释和流程设计。

Codex：工程化平替

当任务需要改文件、跑命令、看 diff、做静态网页或代码工具时，再切到 Codex。

共同安全规则

三步以上先要计划；涉及文件先看范围；改动后要人工检查；API Key 不进文档和 Git。

Git / GitHub 的定位

Git 不是本课主角，只是保存检查点。老师第一天只需要知道：状态、添加、提交、推送、拉取。

git status
git add .
git commit -m "feat: create course homepage"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main

GitHub Pages 是可选展示

如果老师想公开展示个人工作流、课程资料或主页，再用 Pages。它不是最终必交成果。

仓库名：<username>.github.io
入口文件：index.html
Settings -> Pages -> Build and deployment -> Deploy from a branch
分支：main
目录：/root
访问：https://<username>.github.io
VPN 与网络

它解决网络路径问题，不解决账号和权限问题

VPN 像一条加密隧道，可能改变网络出口。它不能凭空给你 API 额度，也不能绕过 GitHub 账号权限。

什么时候讲

下载工具失败、API 访问不稳定、公共 Wi-Fi、学校远程资源访问。

讲到哪里停

讲原理、合规使用和风险，不在课堂现场临时找复杂线路。

常见误解

VPN 不是万能隐身。免费 VPN、来路不明配置和共享账号都可能带来风险。
