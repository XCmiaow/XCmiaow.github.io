# 首页动画流畅度与粒子显著性设计

## 背景

首页黑洞由两套动画共同构成：CSS 负责引力雾、吸积盘、光子环和前景尘埃，Canvas 负责 64 个轨道粒子。当前构图和层次已经确定，但逐粒子的实时 `shadowBlur` 与 `lighter` 合成叠加，以及带大面积模糊的 CSS 表面持续变换，会造成明显的帧时间波动。Canvas 热路径还会重复读取布局并创建临时投影对象。

本轮只优化渲染方式，不重新设计首页。

## 目标

- 保留当前黑洞构图、尺寸、光晕、吸积盘、轨迹和全部景深层。
- 保留 64 粒子上限、1.5 DPR 上限和加色混合，不以降帧、减粒子或降低分辨率换取流畅度。
- 以 60Hz 设备上的稳定 60fps 为目标，消除主线程中可避免的逐帧布局读取和实时模糊。
- 粒子整体更醒目，但仍从属于黑洞主体，不改变页面信息层级。
- 保持隐藏页面、离开视口、页面切换和减少动态效果模式下的现有生命周期行为。

## 非目标

- 不迁移到 WebGL 或引入第三方动画库。
- 不修改首页排版、文案、导航、联系区或黑洞几何位置。
- 不引入按设备降级的视觉质量档位。
- 不顺带重构其他页面动画。

## 方案选择

采用“等效渲染优化”：继续使用现有 DOM/CSS 与 Canvas 2D，实现层保持不变，只替换昂贵的绘制方式。

未采用的方案：

- 只缓存布局：风险低，但无法消除实时模糊与复杂 CSS 重栅格化，收益不足。
- OffscreenCanvas 或 WebGL：性能上限更高，但实现范围、兼容性和视觉偏差风险都更大。

## Canvas 设计

### 几何缓存

黑洞中心和内外半径在布局稳定后不会逐帧变化。初始化及尺寸变化时计算一次 `GravityField`，动画帧只读取缓存值。尺寸变化由 `ResizeObserver` 驱动，并用一个 `requestAnimationFrame` 合并连续通知；无 `ResizeObserver` 时保留窗口 `resize` 回退。

粒子 profile、固定旋转角的正余弦和颜色常量提升到模块级。投影直接写入粒子坐标，避免每粒子每帧创建 `{ x, y }` 对象。

### 等效光晕

移除逐粒子的 Canvas `shadowBlur`。初始化时为暖色和冷色粒子预生成小型径向光晕纹理；动画帧保留原有轨迹和核心，并用 `drawImage` 绘制缓存光晕。轨迹使用宽而低透明度的外层与清晰内核模拟原来的柔光，不再触发实时模糊。

继续使用 `lighter` 合成，保证粒子在重叠处仍有现有的发光累积感。粒子数量、生成速率、轨道和生命周期不变。

### 粒子显著性

最终粒子透明度增加约 20%，并限制在 1 以内。暖色与中近景粒子的光晕更清晰，远景粒子仍保持克制。浅色主题的 Canvas 整体透明度由 0.30 提升到 0.36，不增加绘制次数。

## CSS 合成设计

复杂纹理与动画职责分离：

- `gravity-veil` 外层只动画 `transform` 和 `opacity`，静态渐变与 `blur(23px)` 放到内部伪元素。
- 前后吸积盘外层继续承担现有的位移、旋转、裁切和透明度动画，渐变、遮罩、混合模式与静态滤镜放到内部表面。
- 仅对确实持续动画的外层声明 `will-change: transform, opacity`，避免把整个首页强制提升为合成层。

所有渐变、颜色、模糊半径、动画周期和关键帧数值保持不变，因此黑洞静态外观与运动节奏不变。

## 生命周期与回退

- 页面不可见、Canvas 离开视口或组件销毁时停止 rAF。
- 返回视口后重置时间基准，避免一次超大 delta。
- `prefers-reduced-motion` 继续只绘制确定性的静态粒子帧，并关闭 CSS 动画。
- 主题切换不重建粒子；CSS 负责浅色主题可见度。
- Canvas 或 2D context 不可用时，CSS 黑洞仍完整显示。

## 验证设计

先写会在当前实现失败的回归契约，再修改生产代码。

### 性能契约

- 布局稳定后的动画热路径中，黑洞舞台和场景的 `getBoundingClientRect()` 读取数为 0。
- 正常动画的 rAF 间隔中位数不超过 `max(20ms, 静态基线 × 1.25)`。
- 正常动画的 p95 不超过 `max(34ms, 静态基线 × 2)`，超过 34ms 的帧比例不高于 5%。
- 源码契约禁止在逐粒子绘制函数中重新引入 `shadowBlur`。

### 粒子显著性契约

在 DPR=1、减少动态效果的确定性静态帧中，桌面与移动端均满足：

- alpha energy：9000–18000；
- 强可见像素：120–300；
- 最大 alpha：145–210。

保留 `PARTICLE_LIMIT = 64`，显著性提升不能通过堆数量实现。

### 视觉与生命周期契约

- 保留现有五个视口的居中、尺寸、吸积盘可见率和文案避让检查，并将场景宽度限制为当前计算值 ±2px。
- 内部对比暗色/浅色、桌面/移动端截图；隐藏 Canvas 后核对 CSS 黑洞层次与优化前一致。
- 验证可见时持续绘制、离开视口后停止、返回后恢复、减少动态效果只绘一帧、`pagehide` 后彻底清理。
